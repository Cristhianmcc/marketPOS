No se pudo conectar al servidor

Monterrial POS no pudo iniciar el servidor interno. Esto puede ocurrir si hay un problema con la instalación o la base de datos.

Posibles soluciones:

  • Verifica que la base de datos esté accesible
  • Revisa el archivo .env con las credenciales correctas
  • Reinicia la aplicación
  • Contacta soporte técnico si el problema persiste
